# GuessingGameJava
Este es un juego sencillo de adivinanza desarrollado en Java para practicar conceptos fundamentales de programación y uso de interfaces gráficas básicas

## Objetivo 
- Fortalecer el dominio de estructuras de control en el uso de `while` para el ciclo del juego e `if-else` para la lógica de comparación 
- Identificar metódos de la clase Math para la generación de numeros aleatorios
- Implementar herramientas de escritorio de javax.swing a través de la implementación de ventanas de diálogo interactivas a través de la clase.
- Manejar íconos y su importación mediante la obtención de su URL

## Características
1. El juego genera un número pseudo aleatorio del 1 al 100
2. El usuario debe ingresar un número dentro de ese rango y el juego mencionará si el numero ingresado es mas grande o más pequeño que el numero a adivar
3. En caso de equivocarse el juego mostrará un mensaje y un ícono divertido
4. En caso de ganar el programa mostrará un mensaje con el número de intentos obtenidos 

## Instalación y Ejecución 
1. **Clonar el directorio**

```Bash
git clone https://github.com/Derek-Vergara/GuessingGameJava.git
```
2. **Navegar al directorio**
```bash
cd GuessingGameJava
```
3. **Compilar el programa**
```bash
javac src/*.java
```
4.**Ejecutar**
```bash
java -cp src/main
```
